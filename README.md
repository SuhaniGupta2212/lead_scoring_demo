## Lead Scoring Web Agent – Demo

This is a proof-of-concept web application that simulates a lead generation and prioritization engine for 3D in-vitro therapy platforms.

### Features
- Identification of relevant scientific & industry profiles
- Enrichment with funding, location, and research intent
- Probability scoring (0–100) based on weighted signals
- Searchable and exportable dashboard
-Data shown is representative mock data used to demonstrate scoring logic and workflow.

### Scoring Signals
- Role Fit (Toxicology / Safety): +30
- Recent Liver Toxicity Publication: +40
- Series A/B Funding: +20
- Uses In-Vitro Models: +15
- Hub Location (Boston, Cambridge, Basel): +10

### Future Integrations
- LinkedIn / Sales Navigator via Proxycurl
- PubMed API for scientific publications
- Crunchbase / PitchBook for funding intelligence
- NIH RePORTER for academic grants

This demo focuses on logic, prioritization, and usability rather than live scraping.
